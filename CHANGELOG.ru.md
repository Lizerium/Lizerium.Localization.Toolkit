# Changelog / Журнал изменений

## 1.0.5 - 2026-05-07

### Русский | [Английский](CHANGELOG.md)

#### Исправлено

- Исправлен критичный баг в `Lizerium.Localization.Xaml.Vsix`, появившийся после предыдущего релиза.
- Версия `Lizerium.Localization.Xaml.Vsix` поднята до `1.0.5`; версии остальных библиотек и расширений остаются независимыми.
- NuGet-пакеты подняты до `1.0.5` для исправления упаковки analyzer/code fix.

## 1.0.4 - 2026-05-07

### Русский | [Английский](CHANGELOG.md)

#### Добавлено

- Добавлено отдельное расширение Visual Studio 2022 `Lizerium.Localization.EditorHints`, которое показывает inline-подсказки прямо в C# редакторе для вызовов сгенерированной локализации, например `L.MainWindow.Title()`.
- Подсказки читают значения из `Resources/Localization/Strings*.resx`.
- Язык подсказок берется из языка интерфейса Visual Studio; если подходящего RESX-файла нет, используется английский.
- Подсказки обновляют значения из RESX без переоткрытия C# файла за счет проверки времени изменения `Strings*.resx`.
- Release workflow теперь собирает и публикует оба расширения Visual Studio:
  - `Lizerium.Localization.Xaml.Vsix`
  - `Lizerium.Localization.EditorHints`
- В release assets добавлен этот двуязычный `CHANGELOG.md`.

#### Изменено

- Версия релиза зафиксирована как `1.0.4` для NuGet-пакетов и обоих VSIX-расширений.
- XML documentation в source generator теперь содержит ключи локализации и доступные значения `en` / `ru`, чтобы Quick Info показывал полезный текст даже без editor hints VSIX.
- Упаковка VSIX ужесточена: сборки Visual Studio не попадают внутрь пакета расширения.
- WPF sample демонстрирует вызовы сгенерированного API локализации и inline-подсказки.

#### Исправлено

- Устранены проблемы MEF composition, которые возникали из-за упаковки лишних Visual Studio DLL или небезопасной регистрации editor-layer компонентов.
- Исправлено устаревание подсказок после генерации новых RESX-ключей.
- Исправлено позиционирование подсказок: несколько строк выравниваются в читаемую правую колонку.
- Существующее поведение и версия XAML VSIX сохранены, а новый editor hints VSIX добавлен отдельным расширением.
