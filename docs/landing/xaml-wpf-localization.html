WPF XAML localization with a Visual Studio light bulb

Install `Lizerium.Localization.Xaml.Vsix`, place the caret inside a literal, and convert UI text into `{loc:Loc Key}` with generated RESX entries.

Before

<Window Title="Localization Sample">
  <Button Content="English" />
</Window>

After

<Window
  xmlns:loc="clr-namespace:Lizerium.Localization.Core;assembly=Lizerium.Localization.Core"
  Title="{loc:Loc MainWindow_Title_LocalizationSample}">
  <Button Content="{loc:Loc MainWindow_Button_Content_English}" />
</Window>

AI settings

Use Tools -> Options -> Lizerium Localization -> AI Servers to configure Ollama, model, endpoint, LibreTranslate URL, and fallback behavior.
