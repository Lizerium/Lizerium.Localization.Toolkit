Локализация WPF XAML через лампочку Visual Studio

VSIX преобразует текстовые значения XAML в `{loc:Loc Key}`, добавляет namespace и создает записи в `.resx`.

До

<Window Title="Localization Sample">
  <Button Content="English" />
</Window>

После

<Window
  xmlns:loc="clr-namespace:Lizerium.Localization.Core;assembly=Lizerium.Localization.Core"
  Title="{loc:Loc MainWindow_Title_LocalizationSample}">
  <Button Content="{loc:Loc MainWindow_Button_Content_English}" />
</Window>

Настройки AI

Откройте Tools -> Options -> Lizerium Localization -> AI Servers и задайте Ollama URL, модель, endpoint и LibreTranslate URL.
