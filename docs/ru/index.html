Lizerium.Localization.Toolkit
Старт AI настройки Доки Экосистема Гайды GitHub
RESX локализация .NET с локальным AI

Автогенерация переводов, RESX ключей, C# API и WPF XAML локализации.

Lizerium.Localization.Toolkit объединяет runtime, source generator, Roslyn analyzers, Visual Studio CodeFix, XAML VSIX и локальные AI переводы через Ollama и LibreTranslate.

Начать с NuGet XAML лампочка Настроить AI

Быстрый старт

NuGet пакет

<PackageReference Include="Lizerium.Localization.Toolkit" Version="1.0.0" />

Добавьте `.resx` как `AdditionalFiles` и `Content`, затем используйте сгенерированный API.

VSIX для XAML

src\Lizerium.Localization.Xaml.Vsix\bin\Release\net472\Lizerium.Localization.Xaml.Vsix.vsix

Лампочка Visual Studio заменяет XAML текст на `{loc:Loc Key}` и создает записи в ресурсах.

Популярные сценарии

Как автогенерировать переводы?

AI CodeFix и XAML лампочка создают ключи и переводы автоматически.

Как локализовать WPF XAML?

Преобразуйте `Content`, `Text`, `Title` и выделенный текст в `{loc:Loc Key}`.

Как настроить Ollama?

Настройка локального AI для VSIX, NuGet analyzer и AI Core.
